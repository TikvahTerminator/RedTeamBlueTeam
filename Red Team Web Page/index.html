What is this?

This is the See Sh#rp Insecure Web Server. A web server that is SPECIFICALLY designed to be insecure
and vulnerable against different kinds of attacks to help a user practice their Red Team web skills. However, saying that:
DO NOT EXPOSE THIS SERVER TO AN EXTERNAL NETWORK!
Doing so may be a breach of your company's Security policy, and could put you at risk of attacks!

What can I do on here?
